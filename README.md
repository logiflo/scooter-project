An Electric Scooter Hire app system, created using UML, OOP, JS Classes and TDD.

## Getting Started

```bash
$ npm install
```
## Running Tests
To run tests in "watch" mode

```bash
$ npm test
```
To get a coverage report
```bash
$ npm run test:coverage
```
